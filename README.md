# house-prize-prediction
Overview

This project predicts house prices based on factors such as area, number of bedrooms, and location using the Linear Regression algorithm. Users can enter house details, and the model estimates the expected house price.

Features
Predicts house prices using machine learning.
Takes user input for area, bedrooms, and location.
Uses Linear Regression for prediction.
Simple and beginner-friendly implementation.
Technologies Used
Python
Pandas
NumPy
Scikit-learn
How It Works
Loads housing data.
Trains a Linear Regression model.
Accepts user input.
Predicts and displays the estimated house price.
Sample Input
Enter House Area (sq ft): 2400
Enter Number of Bedrooms: 4
Enter Location Code (1-5): 3
Sample Output
Estimated House Cost: ₹ 70.85 Lakhs
